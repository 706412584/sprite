# 布局控件扩展清单

## 立即可做

- Image
- ProgressBar
- Badge
- Spacer
- 线性布局（LinearLayout）
- 相对布局（RelativeLayout MVP）

这些控件或布局不需要完整的新求解系统，主要工作是：

- 类型定义补齐
- 属性面板补齐
- 预览渲染补齐

## 需要预览器增强

- List
- Tabs / TabPage
- Modal
- Icon
- Avatar
- RichText
- 九宫格图片
- 轮播 Banner

这些能力通常已经有数据结构基础，但需要更细的专门渲染分支。

## 需要新布局算法

- 复杂约束布局
- Anchor / Constraint Layout
- Dock / Split 布局
- Table / Form 布局
- Masonry / 瀑布流
- 虚拟列表
- 可折叠树表

这些能力不能只靠样式字段，需要额外的布局求解或测量阶段。

## 当前优先级建议

1. Image
2. ProgressBar
3. Badge
4. LinearLayout
5. RelativeLayout MVP
6. List
7. Tabs / TabPage

## RelativeLayout MVP 范围

- 对齐父布局：上 / 下 / 左 / 右 / 水平居中 / 垂直居中
- 相对兄弟节点：上方 / 下方 / 左侧 / 右侧
- 对齐兄弟节点边：左对齐 / 右对齐 / 顶对齐 / 底对齐

## LinearLayout MVP 范围

- 横向 / 纵向
- 主轴对齐
- 交叉轴对齐
- 间距 gap
- 与基础 padding / margin 联动
